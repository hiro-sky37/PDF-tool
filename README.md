# Minimal PDF Editor

ブラウザだけで動くミニマルなPDF編集ツールです。
[サイトを開く](https://hiro-sky37.github.io/PDF-tool/)

## 機能

- PDF読み込み
- ページ選択
- 選択ページの左回転 / 右回転
- 選択ページの削除
- ドラッグでページ順を並び替え
- 編集済みPDFを書き出し

## 使い方

1. `index.html` をブラウザで開く
2. PDFを選択またはドラッグ&ドロップ
3. ページをタップして選択
4. 回転・削除・並び替え
5. 「保存」でPDFを書き出し

## 方針

- サーバー不要
- 外部API不要
- 画像変換、結合、分割、圧縮などの余計な機能は未実装
- PDF処理はブラウザ内で完結

## 使用ライブラリ

- PDF.js: サムネイル表示
- pdf-lib: PDFの再構成・回転保存
- SortableJS: スマホでも扱いやすい並び替え
